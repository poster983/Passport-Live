webpack/unsavedWork

Module: webpack/unsavedWork

Members

Methods
  • button
  • changed
  • destroy
  • reset

Browser module for storing if work is unsaved and then notifying the user. This is ment to be used with the back buttons on the mustache mixen pages. If the button is clicked and there is unsaved work, the button will need to be pressed again for the button action to go through.

Initiates a button for tracking if work is unsaved.

The element to initiate for trackeing

Properties
Name Type Attributes Description
onAction function <optional>

called when the button action goes through. IE The page is discarded. The click event and the element are passed
onDiscard function <optional>

called when the button is pressed after being warned. Like callbacks.onAction, but not called if everything is saved.
onWarn function <optional>

called when the button action is canceled and a warning should be showed. The click event and the element are passed
onSave function <optional>

called when the element is saved and it is save to discard. The Element is passed
onReset function <optional>

called when the element is reset to default values. The Element is passed
