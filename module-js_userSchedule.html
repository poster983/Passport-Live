Student Schedule/Student Dashboard Schedule object

Properties
Name Type Attributes Description
teacherID string <nullable>

Holds an account ID linked to this period. Basically, the student is under this teacher's jurisdiction normally during this period. If null, the student is assumed to not have teacher supervision.
Example
{
   "a": {
       "teacherID": "2e75aa94-fa63-461a-be06-5345322bebdf"
   },
   "b": {
       "teacherID": "0115c0b0-ee04-4e88-a21f-d2029601b276"
   },
   "flex": {
       "teacherID": null
   },
   "lunch": {
       "teacherID": null
   }
}

teacherSchedule

Teacher or Supervisor Schedule / Teacher Dashboard Schedule object

accountSchedule.js:61
Properties:
Name Type Description
somePeriodName Object.<...string, Object>

the key for this should be a period constant defined in the configs

Properties
Name Type Attributes Description
className string <optional>

Friendly name for the class being taught or activity being supervised
isTeaching boolean

If true, students will be discouraged from requesting a pass that period.
room string <optional>

room#/name that the user will primarily be in.
passLimit int <optional>

The maximum number of passes to accept per period. Any passes requested over that period will be in a state of "Queue". Setting to 0 will Queue all passes.
Example
{
   "a": {
       "className": "Algebra 1",
       "isTeaching": true,
       "room": "W-2048",
       "passLimit": 0
   },
   "b": {
       "className": "AP Computer Science",
       "isTeaching": true,
       "room": "E-404",
       "passLimit": 0
   },
   "c": {
       "className": "Sub Period",
       "isTeaching": false,
       "room": "W-2048"
   },
   "flex": {
       "isTeaching": false,
       "room": "W-2048",
       "passLimit": 10
   },
   "lunch": {
       "isTeaching": false,
       "room": "Teacher Lounge",
       "passLimit": 0
   }
}
